## 4. bank_account（銀行口座）★MASTERテーブル

銀行口座情報を管理する**MASTERテーブル**。取引番号（torihiki_no）を自動採番。
`customer_id` で顧客（業務テーブル）に紐付く。**1つの顧客が複数の銀行口座を持つ（1:N）**。

| 論理名 | 物理名 | データ型 | PK | NOT NULL | DEFAULT | 説明 |
|--------|--------|---------|:--:|:--------:|---------|------|
| ID | id | BIGSERIAL | ○ | ○ | 自動採番 | 内部主キー |
| 取引番号 | torihiki_no | VARCHAR(20) | - | ○ | 自動採番（torihiki_seq） | 一意の取引番号（BK000001〜） |
| **顧客ID** | **customer_id** | BIGINT | - | - | - | **FK → customer(id) / 1顧客:多口座** |
| 支店番号 | branch_code | VARCHAR(10) | - | - | - | 銀行支店番号 |
| 銀行名称 | bank_name | VARCHAR(200) | - | ○ | - | 銀行名 |
| 口座種類 | account_type | VARCHAR(20) | - | ○ | - | 普通 / 当座 |
| 口座番号 | account_number | VARCHAR(50) | - | ○ | - | 口座番号 |
| 口座名義 | account_holder | VARCHAR(100) | - | ○ | - | 口座名義人 |
| 作成日時 | create_time | TIMESTAMP | - | ○ | NOW() | レコード作成日時 |
| 更新日時 | update_time | TIMESTAMP | - | ○ | NOW() | レコード更新日時 |

**シーケンス:**
- torihiki_seq: START 1, 採番フォーマット `'BK' || LPAD(nextval, 6, '0')` → 例: BK000001, BK000002, ...

**制約:**
- UNIQUE (torihiki_no)
- FOREIGN KEY (customer_id) REFERENCES customer(id) ON DELETE SET NULL
- CHECK (account_type IN ('普通', '当座'))

**インデックス:**
- idx_ba_torihiki (torihiki_no) UNIQUE
- idx_ba_customer (customer_id)

## 8. FK関係図（customer ↔ bank_account / 1:N）

```
┌──────────────────────┐         ┌──────────────────────┐
│      customer         │◄────────│    bank_account       │
│    （業務テーブル）     │  1:N    │   （MASTERテーブル）    │
│                      │         │                      │
│ id (PK)              │         │ id (PK)              │
│ customer_code        │         │ torihiki_no (UNIQUE) │
│ company_name         │         │ customer_id (FK)     │
│ president_name       │         │ branch_code          │
│ ...                  │         │ bank_name            │
│                      │         │ account_type         │
│                      │         │ account_number       │
│                      │         │ account_holder       │
└──────────────────────┘         └──────────────────────┘
  1つの顧客 ← 複数の銀行口座（1:N / ON DELETE SET NULL）
```
