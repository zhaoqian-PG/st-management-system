## 2. リレーションシップ一覧

| 親テーブル | 子テーブル | 関連 | 外部キー | 説明 |
|-----------|-----------|:--:|---------|------|
| employee | attendance | 1 : N | attendance.employee_id | 1人の社員に複数の勤務記録 |
| customer | bank_account | **1 : N** | bank_account.customer_id | **1つの顧客が複数の銀行口座を持つ** |
| customer | invoice | 1 : N | invoice.customer_id | 1つの顧客に複数の請求書 |
| invoice | order_documents | 1 : N | order_documents.invoice_id | 1つの請求書に複数の注文書ファイル |

## 3. テーブル間関連図（簡略版）

```
                    ┌──────────┐
                    │   user   │  (認証独立)
                    └──────────┘

  ┌──────────┐       ┌──────────┐
  │ employee │──1:N──│attendance│
  └──────────┘       └──────────┘

  ┌──────────┐       ┌────────────────────┐
  │ customer │◄──1:N─│  bank_account      │
  │(業務テーブル)│     │  (MASTERテーブル)   │
  └────┬─────┘       │  customer_id (FK) │
       │             └────────────────────┘
                                    │ 1:N
                                    ▼
                             ┌──────────┐       ┌────────────────┐
                             │ invoice  │──1:N──│order_documents │
                             └──────────┘       └────────────────┘
```

## 4. インデックス設計

| テーブル | インデックス名 | カラム | 種別 | 目的 |
|----------|--------------|--------|------|------|
| user | idx_user_username | username | UNIQUE | ログイン認証の高速化 |
| employee | idx_employee_code | employee_code | UNIQUE | 社員番号検索 |
| employee | idx_employee_name | name | INDEX | 氏名検索 |
| employee | idx_employee_department | department | INDEX | 部署別検索 |
| bank_account | idx_ba_torihiki | torihiki_no | UNIQUE | 取引番号検索（MASTERキー） |
| customer | idx_customer_code | customer_code | UNIQUE | 顧客番号検索 |
| customer | idx_customer_company | company_name | INDEX | 会社名検索 |
| customer | idx_customer_torihiki | torihiki_no | INDEX | 取引番号別顧客検索 |
| attendance | idx_att_employee_date | employee_id, work_date | INDEX | 社員×日付の複合検索 |
| attendance | idx_att_date | work_date | INDEX | 日付範囲検索 |
| invoice | idx_invoice_number | invoice_number | UNIQUE | 請求書番号検索 |
| invoice | idx_invoice_customer_ym | customer_id, year, month | INDEX | 顧客×年月の複合検索 |
| order_documents | idx_ordoc_invoice | invoice_id | INDEX | 請求書別注文書一覧 |
