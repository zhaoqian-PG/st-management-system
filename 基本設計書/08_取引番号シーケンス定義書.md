## 2. torihiki_seq（取引番号シーケンス）

| 項目 | 値 |
|------|-----|
| シーケンス名 | `torihiki_seq` |
| 開始値 | 1 |
| 増分 | 1 |
| 最大値 | 999999 |
| サイクル | NO |
| フォーマット | `'BK' \|\| LPAD(nextval, 6, '0')` → BK000001 |
| 使用テーブル | bank_account.torihiki_no |

```sql
CREATE SEQUENCE IF NOT EXISTS torihiki_seq START 1;

-- DEFAULT 定義
torihiki_no VARCHAR(20) NOT NULL UNIQUE
    DEFAULT 'BK' || LPAD(nextval('torihiki_seq')::text, 6, '0')
```

---

## 3. customer_code_seq（顧客番号シーケンス）

| 項目 | 値 |
|------|-----|
| シーケンス名 | `customer_code_seq` |
| 開始値 | 1 |
| 増分 | 1 |
| 最大値 | 9999 |
| サイクル | NO |
| フォーマット | `'CUS' \|\| LPAD(nextval, 4, '0')` → CUS0001 |
| 使用テーブル | customer.customer_code |

```sql
CREATE SEQUENCE IF NOT EXISTS customer_code_seq START 1;

-- DEFAULT 定義
customer_code VARCHAR(20) NOT NULL UNIQUE
    DEFAULT 'CUS' || LPAD(nextval('customer_code_seq')::text, 4, '0')
```

**採番例:**

| nextval 戻り値 | フォーマット後 |
|:------------:|:------------:|
| 1 | CUS0001 |
| 10 | CUS0010 |
| 100 | CUS0100 |
| 9999 | CUS9999（最終値） |

## 4. 運用ルール

| 項目 | torihiki_seq | customer_code_seq |
|------|:-----------:|:-----------------:|
| 採番タイミング | INSERT 時 | INSERT 時 |
| 欠番の扱い | 詰めない | 詰めない |
| 手動設定 | 特殊ケースのみ許可 | 特殊ケースのみ許可 |
| リセット方法 | `ALTER SEQUENCE ... RESTART WITH 1` | 同左 |
