#### 2.1.2 コンポーネント仕様

| UI要素 | Ant Design Component | props |
|--------|---------------------|-------|
| エラーメッセージ | `<Alert>` | type="error", showIcon, message, closable |
| ユーザー名入力 | `<Input>` | prefix=UserOutlined, placeholder |
| パスワード入力 | `<Input.Password>` | prefix=LockOutlined, placeholder |
| ログインボタン | `<Button>` | type="primary", htmlType="submit", block, loading |

#### 2.1.3 イベント仕様

| イベント | 発生条件 | 処理内容 |
|----------|---------|---------|
| onFocus (Input) | 入力フィールドにフォーカス | エラーメッセージクリア (setErrorMsg('')) |
| onChange (Input) | 入力値変更 | 対応する state を更新 |
| onSubmit (Form) | 「ログイン」ボタン クリック | バリデーション → auth.login() → 成功時 navigate('/employee') |
| onKeyDown (Form) | Enter キー押下 | onSubmit と同じ処理 |

#### 2.1.4 バリデーション仕様

| フィールド | ルール | メッセージ(ja) | メッセージ(zh) |
|-----------|--------|---------------|---------------|
| username | required | ユーザー名を入力してください | 请输入用户名 |
| password | required | パスワードを入力してください | 请输入密码 |

#### 2.1.5 エラー処理

| エラー種別 | HTTP status | 表示メッセージ | ユーザー操作 |
|-----------|:----------:|--------------|------------|
| 認証失敗 (ID/PW不一致) | 401 | ユーザー名またはパスワードが間違っています。 | 再入力 |
| ネットワークエラー | - (Network Error) | サーバーに接続できません。ネットワークを確認してください。 | リトライ |
| サーバーエラー | 500 | サーバーエラーが発生しました。しばらく経ってからお試しください。 | リトライ |

### 2.7 SCR-BANK — 銀行口座管理画面（専用ページ）

**画面ID**: SCR-BANK
**URL**: `/bank-accounts`

銀行口座情報は機密情報のため、bank_account 主テーブルとして独立した専用ページで管理。ポップアップやモーダルは使用しない。

**画面構成**:

```

  業務テーブル: [ドロップダウン（顧客選択）]          [+ 口座を追加]

⚠ セキュリティ注意喚起バナー（黄色）

口座一覧カード（主テーブル）:
  Table: # │ 銀行名 │ 支店名 │ 口座種類 │ 口座番号(****4321形式) │ 口座名義 │ 紐付く顧客 │ 操作
  └── 口座番号横 [👁 表示] → 確認後表示、30秒自動再マスク
  └── 行クリック → 顧客業務情報（業務テーブル）を下に表示
  └── 操作: [編集] [削除]

顧客業務情報カード（条件表示）:
  会社名 / 社長名 / Webサイト / 住所 / 営業担当(氏名/連絡先/メール) / 事務担当(氏名/連絡先/メール)
  [🏢 顧客管理画面で編集] ボタン

  銀行名* / 支店名* / 口座種類* / 口座番号*(password型) / 口座名義*
  [キャンセル] [保存] (100px幅統一・右下)
```

**遷移元**:
- サイドバー「🔒 銀行口座管理」メニュー
- 顧客編集モーダル内「🔒 銀行口座管理画面へ」ボタン（モーダル自動クローズ＋本画面に遷移）

※ ポップアップ/モーダル不使用。bank_account 主テーブルの専用ページで機密情報を安全に管理。
※ 口座番号はデフォルトマスク、表示には確認操作が必要。
※ 行クリックで紐付く顧客（業務テーブル）の参照情報を表示。

#### 2.5.3 CSVエクスポート詳細

| 項目 | 仕様 |
|------|------|
| エンコーディング | UTF-8 BOM付き (Excel日本語表示対応) |
| 区切り文字 | カンマ (,) |
| ヘッダー行 | 日付,社員名,勤務時間(h),残業時間(h),ステータス,備考 |

```java
// CsvExportUtil.java
public static void writeCsv(HttpServletResponse response,
    List<AttendanceDTO> data, int year, int month) throws IOException {
    response.setContentType("text/csv; charset=UTF-8");
    response.setHeader("Content-Disposition",
        "attachment; filename=\"勤務記録_" + year + String.format("%02d", month) + ".csv\"");
    // BOM for Excel
    response.getOutputStream().write(0xEF);
    response.getOutputStream().write(0xBB);
    response.getOutputStream().write(0xBF);

    try (PrintWriter writer = response.getWriter()) {
        writer.println("日付,社員名,勤務時間(h),残業時間(h),ステータス,備考");
        for (AttendanceDTO dto : data) {
            writer.printf("%s,%s,%.1f,%.1f,%s,\"%s\"%n",
                dto.getWorkDate(), dto.getEmployeeName(),
                dto.getWorkHours(), dto.getOvertimeHours(),
                dto.getStatus(),
                dto.getRemark() != null ? dto.getRemark() : "");
        }
    }
}
```
